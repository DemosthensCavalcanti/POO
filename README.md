# Programação Orientada a Objetos – UFPB

Repositório criado para acompanhar a disciplina de POO.  
Aqui ficam as atividades, exercícios, anotações e implementações feitas ao longo do semestre.

## Estrutura

- **src/** código-fonte organizado em pacotes.
- **docs/** materiais da disciplina e diagramas UML.
- **examples/** pequenos trechos de código usados para testar conceitos.
- **assets/** imagens ou recursos usados no projeto.

## Tecnologias usadas

- Java
- VS Code
- Git e GitHub

## Objetivo do repositório

Manter o código organizado e documentado enquanto pratico:

- encapsulamento
- herança
- polimorfismo
- interfaces
- exceções
- boas práticas de organização de classes

## Como executar

1. Certifique-se que tem o Java instalado (versão pedida na disciplina).
2. Abra o repositório no VS Code.
3. Compile e execute a partir da pasta `src` usando a extensão de Java ou o terminal integrado.
